{% blocktrans %} PyCon is a conference by the Python community for the community. It is for Python enthusiasts of all experience levels, from new users to core developers. The original PyCon was formed in North America in 2003, but there are now a number of other conferences being run in the PyCon spirit around the world. -- PyCon: Community-Run Python Conference {% endblocktrans %}

{% blocktrans %} PyCon APAC/TW 2014 is organized for pythonistas in Asia-Pacific and Taiwan. It will be hosted in Taipei, and we would like to invite you to join this big event, being a part of it by sponsoring, promoting your brand and product, and even making your company succeed through it. Two previous PyCon TW (2012 and 2013) was formed with support of the following, but not limited to, active user groups: {% endblocktrans %}

  • PyHUG - Python Hsinchu User Group
  • Taipei.py - Taipei Python User Group
  • Tainan.py - Python Tainan User Group
